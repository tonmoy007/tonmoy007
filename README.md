## 👨‍💻 About Me

**Lead Software Engineer** @ [Polygon Technology](https://polygontechnology.io/) (Bangladesh)

I build **deterministic infrastructure for AI agent workflows** — gated SDLC pipelines that make agent-driven development auditable, replayable, and production-safe. My work spans the full stack: from low-level orchestration engines to high-throughput scraping pipelines.

- 🔭 Currently building **[forge-os](https://github.com/tonmoy007/forge-os)** — a local-first, kernel-agnostic SDLC orchestration CLI with 12-stage quality-gated lifecycle
- 🧩 Creator of **[forge-plugins](https://github.com/tonmoy007/forge-plugins)** — gated SDLC orchestrator for Claude Code (REQ-ID traceability, cost-capped background agents)
- 📖 Curating **[religious-study](https://github.com/tonmoy007/religious-study)** — a comprehensive Obsidian vault for comparative religious research with full scriptures across 12+ traditions
- 🤖 Building **bAss** — a banking chatbot platform (FastAPI/Docker/CrewAI/NATS/PostgreSQL)
- 🌱 Deepening into **MCP protocol**, **token-efficient agents**, and **OTLP tracing** for agent systems
